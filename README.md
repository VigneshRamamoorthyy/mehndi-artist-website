# Rekha Mehndi Artist Website

A premium and responsive landing page built for a professional mehndi artist to showcase designs, attract clients, and simplify customer inquiries.

## Live Website

🔗 https://rekhamehndi.vercel.app

## Overview

This project was built using React, Vite, and Tailwind CSS with a luxury bridal-inspired design. The website is optimized for mobile, tablet, and desktop users and focuses on providing a premium user experience.

## Features

* Premium bridal-themed design
* Fully responsive layout
* Hero section with WhatsApp booking CTA
* Service showcase
* Mehndi gallery
* About section
* Testimonials section
* Contact section
* Floating WhatsApp button
* Google Analytics integration
* Vercel Analytics integration
* Fast deployment using Vercel

## Services Displayed

* Bridal Mehndi
* Engagement Mehndi
* Festival Mehndi
* Guest Mehndi
* Baby Shower Mehndi

## Installation

Clone the repository:

```bash
git clone https://github.com/VigneshRamamoorthyy/mehndi-artist-website.git
```

Navigate to project:

```bash
cd mehndi-artist-website
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
